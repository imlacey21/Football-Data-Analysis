# Football-Data-Analysis
Below are som really useful resources with regards to Football Analysis/Data

https://www.youtube.com/@McKayJohns - One of the best series of tutorials with regards to Football Viz's & Web Scraping
https://github.com/griffisben/Soccer-Analyses - Ben is an industry pro and really forthcoming with tips, advice, tutorials etc. Also makes really interesting apps for Radars etc
https://github.com/devinpleuler/analytics-handbook - Really helpful to work through
https://mplsoccer.readthedocs.io/en/latest/installation.html - Lots of code examples & documentation
https://www.liamhenshaw.com/ - Definitely one of the best personalities in this sphere, has no problem with sharing industry information and sharing useful tips and resources
